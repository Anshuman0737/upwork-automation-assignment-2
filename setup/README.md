## Setup Instructions

### Prerequisites
- Docker & Docker Compose
- Apify account
- OpenAI API key
- Airtable base and API key

### Run n8n with Docker
1. Clone this repository
2. Navigate to the project directory
3. Run:
   docker compose up -d
4. Access n8n at http://localhost:5678
5. Import the workflow JSON from the workflow folder
6. Configure credentials using n8n Credentials Manager
